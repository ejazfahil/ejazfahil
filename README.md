## 💡 About Me

I'm a **Data Scientist & AI Solutionist Engineer** building production-grade ML systems that turn complex data into real-world impact. My work spans:

- 🤖 **RAG & Agent Systems** — stateful retrieval agents with human-in-the-loop gates
- 🦙 **LLM Fine-Tuning** — QLoRA, PEFT, AWQ quantization, vLLM serving
- 🔍 **Retrieval Research** — benchmarking embedding-free paradigms (BM25, ColBERT, hybrid)
- 🏭 **Industrial AI** — OCR pipelines, sensor anomaly detection, predictive maintenance
- 📊 **MLOps** — Prometheus/Grafana observability, Docker, CI/CD, PostgreSQL/pgvector

> **Currently building:** A three-part production MLOps blueprint — RAG agent + QLoRA pipeline + vectorless-RAG benchmark.
